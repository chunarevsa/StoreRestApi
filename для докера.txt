Запустим MySQL в контейнере и подключимся к volumes и network, 
	которые мы создали выше. 
Докер извлекает изображение из концентратора и запускает его локально.

docker run -it --rm -v mysql_data:/var/lib/mysql -v mysql_config:/etc/mysql/conf.d --network mysqlnet1 --name mysqlserver -e MYSQL_ROOT_PASSWORD=root -e MYSQL_DATABASE=websitechsa -p 3306:3306 mysql:8.0.23

Теперь давайте запустим наш контейнер. 
На этот раз нам нужно установить переменную среды MYSQL_URL, 
чтобы наше приложение знало, какую строку подключения использовать 
для доступа к базе данных. Мы сделаем это с помощью команды docker run.

docker run --rm --name springboot-server --network mysqlnet1 -e MYSQL_URL=jdbc:mysql://mysqlserver/websitechsa -p 8088:8088 java-docker
